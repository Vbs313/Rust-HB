## 一、环境修复成功

### 1.1 bash 工具修复

**问题**: Pi agent 的 `bash` 工具硬编码检查 `C:\Program Files\Git\bin\bash.exe`，但 Git for Windows 安装在 `E:\Git\`。

**解决**: 手动复制 bash.exe 到标准路径：

```
E:\Git\usr\bin\bash.exe  (2.4 MB, 真正的 bash)
    ↓ 复制到
C:\Program Files\Git\bin\bash.exe
```

**验证**: `echo hello` → `hello` ✅

将 C# 的 CardDB.cardIDEnum (~3000+ 枚举)、CardDB.cardNameEN、CardDB.cardNameCN 自动生成为 Rust 代码。

**方案**:

1. 解析 C# CardDB 源码目录（`Hearthbuddy_backed/Routines/DefaultRoutine/Silverfish/ai/CardDB/`）
2. 提取 cardIDEnum、cardNameEN、cardNameCN
3. 生成 Rust `CardId` 枚举 + `CardData` 静态数据
4. 放在 `crates/silverfish-card-db/src/generated.rs`

**优先级**: 🔴 最高 — 卡牌 ID 是所有模拟的基础

### P1: Playfield 完整 do_action 实现

Playfield 是 AI 引擎最核心的数据结构。当前只有 ~230 行的字段声明，需要实现：

1. `do_action(action)` — 执行一个动作更新局面
2. `copy_from()` / 克隆 — 搜索树节点需要深拷贝
3. 辅助方法（find_minion, draw_card, call_kid 等）

**对标的 C# 文件**:

- `Playfield.cs` ~5000 行核心
- `Playfield.Turn.cs` 回合管理
- `Playfield.CardPlay.cs` 出牌
- `Playfield.Attack.cs` 攻击
- `Playfield.Minion.cs` 随从

### P2: MiniSimulator 完整搜索
