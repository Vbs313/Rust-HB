### 卡牌数据结构 (Rust)

```rust
#[derive(Debug, Clone, Copy, PartialEq)]
#[repr(C)]
pub struct CardData {
    pub id: u32,            // 卡牌 ID（Hearthstone 官方 ID）
    pub type_id: u8,        // 1=随从, 2=法术, 3=武器, 4=英雄...
    pub cost: i32,          // 费用
    pub attack: i32,        // 攻击力
    pub health: i32,        // 生命值
    pub armor: i32,         // 护甲值
    pub durability: i32,    // 耐久度
    pub set_id: u32,        // 扩展包 ID
    pub name_cn: &'static str,  // 中文名
    pub name_en: &'static str,  // 英文名
    pub text: &'static str,     // 卡牌描述
}
```

### 查找性能

- 数组按 `id` 升序排列 → **二分查找 O(log n)**
- 所有数据编译期为 `&'static` 引用 → **零运行时解析、零内存分配**
- 不重复数据，无 HashMap 开销
