## 四、与原版 C# 项目的质量对比

| 维度           | C# 原版                 | Rust 版                 | 改进               |
| -------------- | ----------------------- | ----------------------- | ------------------ |
| **类型安全**   | ❌ 大量 object 装箱     | ✅ 泛型 + 强类型枚举    | 编译期类型检查     |

## 五、环境限制

### 5.1 Bash 工具不可用

- **根因**: Git for Windows 安装在 `E:\Git\`（非标准 `C:\Program Files\Git\`）
- **Pi agent 的 bash 工具**硬编码检查 `C:\Program Files\Git\bin\bash.exe`
- 当前用户无管理员权限写入 `C:\Program Files`
- **权宜方案**: 用 `subagent delegate` 配合 `cmd.exe` 执行命令

### 5.2 已验证的可用替代

| 方式                              | 可用?                                | 用途             |
| --------------------------------- | ------------------------------------ | ---------------- |
| cmd.exe（通过 subagent）          | ✅ Yes                               | 编译器、文件操作 |
| WSL                               | ❌ 未安装（wsl.exe 存在但无 distro） | —                |
| Node.js (C:\Program Files\nodejs) | ✅ 可通过 cmd 调用                   | 脚本处理         |
| PowerShell                        | ✅ 可通过 cmd 调用                   | 高级操作         |
