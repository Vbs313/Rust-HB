# 环境准备与 Rust 工具链安装

> 状态: 🔄 进行中

---

## 一、问题诊断

当前系统缺少以下关键组件：

1. **Git for Windows (bash.exe)** — Pi agent 的 bash 工具依赖于 `C:\Program Files\Git\bin\bash.exe`
2. **Rust 工具链** — 需要安装 rustup + i686-pc-windows-msvc target
3. **Visual Studio Build Tools** — 32 位 C++ 链接器需要 MSVC 工具链

## 二、安装计划

### 2.1 安装 Git for Windows

**方法**: 从 winget 或官网安装

- 安装路径: `C:\Program Files\Git\bin\bash.exe`
- 版本选择: 最新稳定版
- 组件: Git Bash、Git LFS

安装后 Pi agent 的 bash 工具即可正常使用。

### 2.2 安装 Rust 工具链

**步骤**:

1. 安装 `rustup-init.exe`
2. 添加 `i686-pc-windows-msvc` target
3. 验证 `cargo build --target i686-pc-windows-msvc`

### 2.3 安装 Visual Studio Build Tools

**所需工作负载**:

- "使用 C++ 的桌面开发" (Desktop development with C++)
- 包含 `x86` 链接器

---

## 三、执行顺序

```
安装 Git for Windows
    ↓
安装 Rust (rustup)
    ↓
rustup target add i686-pc-windows-msvc
    ↓
安装 VS Build Tools (如需要)
    ↓
验证 cargo build --target i686-pc-windows-msvc
    ↓
初始化 Cargo 工作区
```
